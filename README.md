# 📝 Note-Taking App

A simple and elegant note-taking application built with HTML, CSS, and JavaScript. Create, edit, and delete notes that persist in your browser's local storage.

## Features

- **Create Notes** - Add new notes with a single click
- **Edit Notes** - Update existing notes easily
- **Delete Notes** - Remove notes with confirmation prompt
- **Persistent Storage** - Notes are saved in browser's localStorage
- **Clean Interface** - Simple and intuitive design
- **Real-time Updates** - Notes appear immediately after adding/editing

## Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage API
